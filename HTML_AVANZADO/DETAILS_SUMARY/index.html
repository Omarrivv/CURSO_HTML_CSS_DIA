¿Que es HTML?

HTML son las siglas de HyperText Markup Language, que puede traducirse como Lenguaje de Marcas o Marcado de Hipertexto.

El lenguaje HTML fue creado en 1991 por Tim Berners-Lee, el cual es el fundador de la World Wide Web (WWW). Tim Berners-Lee, es un científico de la computación británico, nacido en Londres el 8 de junio de 1955, conocido por su trabajo en la creación de la World Wide Web, el sistema de navegación que permite a los usuarios acceder a información en la web.

HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

HTML es un estándar a cargo del World Wide Web Consortium (W3C), organismo internacional que se encarga de establecer las normas que debe seguir el lenguaje para que todos los navegadores web puedan interpretar correctamente sus etiquetas.

¿Que es CSS?

CSS son las siglas de Cascading Style Sheets, que puede traducirse como Hojas de Estilo en Cascada. El lenguaje CSS se encarga de la presentación de documentos HTML, es decir, determina el color, forma, diseño y espaciado de los elementos de un documento HTML.

CSS fue desarrollado por el W3C (World Wide Web Consortium) en 1996 por una razón muy sencilla, la necesidad de separar el contenido de la presentación en documentos HTML.

CSS es un lenguaje basado en reglas, con el cual se define que elementos son los que se van a ver afectados, y a que propiedades de estilo van a ser aplicadas.
